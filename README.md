# umi3 project

## Getting Started

Install dependencies,

```bash
$ yarn or npm install
```

Start the dev server,

```bash
$ yarn start or npm start
```

Build project and Local server run

```bash
$ npm run build
$ node ./server.ts
```

## 开发

新建页面

可以自己手动创建，也可运行命令

```bash
$ npm run g
# container是page
# component是component
```
